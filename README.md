# Proyecto QA: Pruebas Funcionales, API Testing (Postman) y Auditoría Lighthouse

##  Descripción del Proyecto
Este repositorio reúne el ecosistema completo de pruebas funcionales y no funcionales realizado sobre la plataforma **demo.guru99.com** y la API pública **PokeAPI**. El proyecto aborda el ciclo de vida completo de un QA Manual: desde el diseño y ejecución de casos de prueba tradicionales de caja negra y el reporte formal de defectos, hasta la verificación de servicios web de tipo REST (Postman) y el análisis técnico comparativo de rendimiento, accesibilidad y SEO mediante Google Lighthouse.

---

##  Tecnologías y Herramientas Utilizadas
* **Diseño y Gestión de Pruebas Manuales:** Microsoft Excel.
* **Seguimiento y Registro de Defectos:** Reportes de Bugs.
* **API Testing:** Postman (Validación de métodos HTTP, endpoints, parámetros y códigos de estado).
* **Auditoría Técnica No Funcional:** Google Lighthouse (Análisis comparativo Desktop vs Mobile documentado).
